# Push to Apify - Quick Reference

## 3 Commands to Deploy

### 1. Install Apify CLI
```bash
npm install -g apify-cli
```

### 2. Login to Apify
```bash
apify login
```
(Opens browser - log in with your Apify account)

### 3. Push to Apify
```bash
cd c:\Users\user\Desktop\create\ai-automation-nocode\google-maps-scraper
apify push
```

That's it! Your Actor will be deployed.

## After Push

1. Go to https://console.apify.com
2. Find "google-maps-scraper" in your Actors list
3. Click "Run" to test it
4. Configure search queries and filters
5. Click "Start" to execute
6. View results in the Dataset
